Gitのインストール

AtCoderを始めたくなり折角なのでGitHubでソース管理しようという気が湧いてきたので環境を構築してみた

Git初心者必須コマンド

  • 1:git clone https:extra リポジトリローカルに置く
  • 2:git add file.extra ソースを変更するためのステージに移動イメージ
  • 3:git commit -m "message" 変更を「確定して履歴に記録する」操作
  • 4:git push ローカルの変更をリモートリポジトリに送信
  • 5:git status 状態の確認
  • 6:git log logを見るコマンド
